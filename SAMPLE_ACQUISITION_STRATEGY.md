# 1688「京建サンプルボックス」調達・送付戦略 (Sample Acquisition Strategy)
**Kyoken Supply - 日本国内パートナー開拓用**

日本のお客様（内装会社・工務店）に品質を確認いただくための「実物サンプル」を、1688を活用して効率的に調達・送付するプロセスです。

---

## 1. サンプルの選定と調達方法 (Procurement)
1688の多くのサプライヤーは「拿样（サンプル注文）」に対応しており、後の本発注時にサンプル代金が返金される仕組みがあります。

| 製品 | 1688での調達形態 | 推奨サンプルサイズ | 備考 |
| :--- | :--- | :--- | :--- |
| **ホーローパネル** | 色見本冊子 (色卡/样册) | 10cm x 15cm 程度 | 重いため小さなピースを推奨 |
| **吸音パネル** | カラーサンプルキット | 全色入りの小型チップ | 2026年の主力製品として全色必須 |
| **定制窗帘** | 布地サンプル (面料卡) | A4サイズ程度のスウォッチ | 遮光・防炎性能を確認できるもの |
| **シームレス壁紙** | 素材・テクスチャ見本 | 各素材 (不織布/布) の断片 | 印刷の質感を確認 |
| **サイン・看板** | 端材サンプル | 10mm厚アクリル / PVC | 切断面の処理クオリティを確認 |

## 2. 集約・検品プロセス (Consolidation)
中国国内の提携倉庫（またはフォワーダー）にて、複数のサプライヤーから届くサンプルを一つにまとめます。

1.  **一括配送：** 1688で各工場から中国国内倉庫へ発送。
2.  **同梱作業 (Kitting)：** 個別の袋に入ったサンプルを、Kyokenオリジナルの「サンプルボックス」に詰め替え。
3.  **独自ラベル貼付：** 弊社ブランドのステッカーを各サンプルに貼付（工場名ではなく、Kyokenの型番で管理）。
4.  **検品：** 日本の基準に照らして、目立つ傷や汚れがないか確認。

## 3. 日本への発送 (Shipping to Japan)
*   **配送方法：** 佐川グローバル / EMS / OCS（2-4日で到着）。
*   **関税対応：** 「Commercial Sample（商品サンプル）」として申告し、関税を最小限に抑制（DDPモデルのシミュレーションとしても活用）。

## 4. 日本国内での展開 (Utilization)
*   **対面提案：** 内装会社への営業時に直接持参。
*   **郵送貸出：** 検討中のお客様に対し、LINE経由で「サンプル請求」を受け付け、レターパック等で送付。
