# 动态调价与利润计算器 (Dynamic Pricing Tool)
**项目：京建サプライ (Kyoken Supply)**

为了应对汇率波动，请让 Agent 使用当前报价体系定期复核价格。此文件已更新为 2026-06-10 之后的规则：Alibaba.com 是出口报价主参考，1688 只作为中国内销底价辅助参考，不能再作为日本客户报价公式的主依据。

---

## 1. 核心计算公式 (The Formula)
**最终日元报价 (JPY Quote) = Alibaba.com出口供应商报价 + 中国国内物流/出口包装 + 国际运输 + 日本进口税费 + 日本国内配送 + 测量/安装/现场确认 + 京建服务费**

*   **当前设定汇率**: 按报价当天 USD/JPY、CNY/JPY 更新，并保留汇率波动余量。
*   **出口参考源**: Alibaba.com，优先看可出口、可定制、MOQ 合理、包装方式清楚的供应商中位价。
*   **1688 作用**: 只判断中国内销底价和成本空间，不直接推导日本客户报价。
*   **物流/税费**: 海运、空运、DDP、日本国内配送、关税、进口消费税按案件条件确认。
*   **服务费**: 测量、安装、现场确认、设计支持、售后协调按品类和现场复杂度加入。

## 2. 自动化更新任务 (Auto-Update Task)
请接手的 Agent 按照以下频率执行：

### [每周一执行]
1.  查询当前 **USD/JPY** 与 **CNY/JPY** 实时汇率。
2.  按 `pricing-system/alibaba-scan-template.csv` 扫描 Alibaba.com 出口供应商报价。
3.  如有必要，用 1688 作为内销底价对照，但不得用 1688 低价直接覆盖网站报价。
4.  对照 `pricing-system/kyoken-pricing-system-v0.1.md` 的成本模块复核。
5.  如需更新页面价格，先保留 Accio 当前窗帘/WPC 基准逻辑，再按品类调整。

## 3. 快速报价参考表 (示例)
| Alibaba.com出口报价 | 需追加确认 | 京建报价判断 |
| :--- | :--- | :--- |
| EXW/FOB 商品价 | 包装、国际运输、税费、日本配送 | 不能直接等于客户价 |
| DDP 供应商价 | 日本国内搬入、测量、安装、现场确认 | 可作为项目价参考 |
| 1688 内销价 | 出口包装、通关、海外物流、售后 | 仅作底价对照 |

---
**提示：若日元大幅贬值，先复核 Alibaba.com 出口报价和物流税费，再判断是否调整网站报价。不要用单一 1688 价格自动覆盖页面。**
