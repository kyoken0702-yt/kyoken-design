# 动态调价与利润计算器 (Dynamic Pricing Tool)
**项目：京建サプライ (Kyoken Supply)**

为了应对汇率波动，请让 Agent 使用此公式定期更新价格。

---

## 1. 核心计算公式 (The Formula)
**最终日元报价 (JPY Quote) = (1688原价 + 国内运费) * 汇率 * (1 + 物流Buffer) * (1 + 利润率)**

*   **当前设定汇率**: 1 CNY = 22.0 JPY (建议留 0.5 的波动空间)
*   **物流 & 关税 Buffer**: 30% (覆盖海运、双清、到门配送)
*   **目标利润率**: 40% (B2B 批发建议毛利)

## 2. 自动化更新任务 (Auto-Update Task)
请接手的 Agent 按照以下频率执行：

### [每周一执行]
1.  查询当前 **CNY/JPY** 实时汇率。
2.  访问 `INTERNAL_OPERATIONS_GUIDE.md` 中的所有 1688 链接。
3.  如有价格变动，根据上述公式重新计算并更新。
4.  将更新后的价格同步到 `index.html` 的产品卡片中。

---
