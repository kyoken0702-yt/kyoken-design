# 京建サプライ 报价体系 v0.1

更新日：2026-06-10

## 1. 报价原则

京建的报价不能只看 Alibaba.com 商品价。对日本客户可交付的价格，应按「中国采购 + 出口包装 + 国际运输 + 日本进口税费 + 日本国内配送 + 现场安装/测量 + 京建服务费」来计算。

采购参考源优先级：

1. Alibaba.com：主参考源。面向海外买家，适合判断出口供应能力、MOQ、FOB/EXW 条件、外贸包装和工厂沟通成熟度。
2. 工厂直接询价：正式报价源。用于确认规格、包装、生产周期、出货条款和样品费用。
3. 1688：辅助参考源。用于判断中国内销底价和成本空间，不直接作为日本客户报价基础。

前台表达：

- 案件ごとにお見積り
- 仕様確認後に概算見積をご案内
- 配送方法・輸入諸費用は案件ごとに確認
- 施工・取付は内容により個別見積

内部计算：

```text
客户报价 = 到岸交付成本 + 日本国内服务成本 + 风险预备金 + 京建服务费/利润
```

## 2. 标准成本结构

### A. 中国侧成本

| 项目 | 说明 | 计价方式 |
|---|---|---|
| Alibaba.com 供应商报价 | 出口供应商报价，不作为客户价直接使用 | USD/RMB，按件/米/㎡/套 |
| 1688 内销参考价 | 仅用于判断中国内销成本空间 | RMB，按件/米/㎡/套 |
| 中国国内运费 | 工厂到集货仓/出口仓 | 实报实销或采购价 3%-8% 预估 |
| 包装费 | 纸箱、护角、珍珠棉、木架、木箱 | 按材质和破损风险 |
| 打样/制版/设计处理 | 图纸确认、文件整理、打样 | 案件计费 |
| 验货/QC | 数量、尺寸、外观、包装确认 | 小单固定费，大单按比例 |
| 中国侧操作费 | 供应商沟通、规格确认、出货协调 | 京建内部服务成本 |

### B. 国际运输

| 方式 | 适用商品 | 特点 | 报价处理 |
|---|---|---|---|
| 国际快递/空运 | 小件、急件、样品、轻货 | 快、贵，按实重/体积重取高 | 单独报价 |
| 海运散货 LCL | 板材、窗帘、标识、批量货 | 适合多数 B2B 案件 | 按 CBM + 港杂费 |
| 整柜 FCL | WPC、大批板材、工程批量 | 单价低，但数量门槛高 | 项目报价 |
| 日本专线/DDP | 小批量测试、轻中货 | 方便，但需确认是否含税费 | 仅作为方案之一 |

### C. 日本进口税费

| 项目 | 说明 | 处理方式 |
|---|---|---|
| 关税 | 按 HS 编码、材质、用途、原产地确定 | 报价前按品类预留，正式按通关确认 |
| 日本消费税 | 进口时通常按货值、运费、关税等基础计算 | 内部按 10% 预留 |
| 通关手续费 | 报关行/快递/货代收取 | 小单固定费，大单按票 |
| 港杂/仓储/换单 | 海运到港后可能发生 | LCL/大货必须预留 |

### D. 日本国内交付

| 方式 | 适用 | 风险 |
|---|---|---|
| 宅配便 | 小件、单箱 200 サイズ以内、30kg 以内 | 尺寸超限不可用 |
| 路线便 | 板材、长件、多箱 | 住宅配送、再配達、时间指定有限制 |
| チャーター便 | 大型板材、工程货、指定现场 | 成本高但稳定 |
| 现场搬入 | 商业设施、楼层搬运、无电梯 | 必须另算 |

### E. 日本现场服务

| 项目 | 标准处理 |
|---|---|
| 上门测量 | 可作为套餐或选项 |
| 标准简易安装 | 仅限既有条件满足、低风险作业 |
| 复杂安装 | 下地补强、特殊固定、拆旧、开孔等另算 |
| 高空作业 | 脚立、足場、吹き抜け、楼梯附近另算 |
| 电气相关 | PSE、电源、配线、结线必须个别确认 |

## 3. 通用计算公式

### 3.1 人民币采购侧

```text
中国侧小计 RMB =
Alibaba.com供应商报价
+ 中国国内运费
+ 包装费
+ 打样/制版/文件处理
+ QC/验货费
+ 中国侧操作费
```

### 3.2 到岸成本

```text
到岸成本 JPY =
中国侧小计 RMB × 汇率 × 汇率缓冲
+ 国际运费 JPY
+ 日本港杂/通关手续费 JPY
+ 关税 JPY
+ 进口消费税 JPY
```

建议内部汇率缓冲：`1.03 - 1.08`

### 3.3 客户报价

```text
材料交付报价 =
到岸成本
+ 日本国内配送
+ 破损/售后预备金
+ 京建服务费
+ 利润
```

```text
含现场服务报价 =
材料交付报价
+ 上门测量
+ 标准安装
+ 复杂施工加价
+ 高空/夜间/商场入馆等特殊费用
```

## 4. 京建推荐报价层级

| 层级 | 对外名称 | 包含内容 | 适用客户 |
|---|---|---|---|
| S1 | 材料供给报价 | 产品 + 包装 + 日本指定地配送 | 施工業者、看板店 |
| S2 | DDP 到仓/到场报价 | 产品 + 出口 + 国际运输 + 进口税费 + 日本配送 | 不想处理进口的客户 |
| S3 | 现场测量报价 | S2 + 上门测量 | 窗帘、板材、内装 |
| S4 | 标准安装套餐 | S3 + 标准简易安装 | 民泊、住宅、店舗小改装 |
| S5 | 工程项目报价 | 定制供应链 + 现场管理 + 多批次配送 | 大批量、店舗连锁、工程 |

## 5. 品类报价规则

### 5.0 已上线 Accio 基准价

窗帘和 WPC 塑木板当前页面价格为 Accio 已生成并上线过的报价，应作为京建首版对外价格基准。

后续 Alibaba.com 扫描的目的不是重新定价，而是：

- 校验该价格下的采购成本空间
- 找出可稳定供货的 3-5 家候选工厂
- 确认出口包装、贸易条款、MOQ、样品费、生产周期
- 反推国际物流、日本配送、安装后是否仍有合理毛利
- 判断哪些价格只能作为「～」起步价，哪些需要改为案件报价

不要用 Alibaba.com 或 1688 的最低价推翻当前页面报价。

- 日本现场测量 + 标准简易安装：按区域、窗数、人员工时计
- 售后预备金：2%-4%

### 5.2 アクリルサイン / ロゴ文字

计价基础：

- 材质：アクリル、金属调、UV印刷
- 厚度、尺寸、颜色、安装方式
- 是否需要孔位、模板、化妆ビス

内部加价重点：

- 易刮花，必须加强保护膜、护角、厚纸板
- 小件可快递，大件建议木架/木箱
- 安装若涉及打孔、下地、水平定位，要另算

建议预备：

- 包装：采购价 8%-18%
- 破损/重做预备：3%-6%
- 关税预留：塑料制品方向先按 0%-4% 估算，正式按 HS 确认

### 5.3 ライトボックス画面 / メニュー画面

计价基础：

- 是否只做画面/フィルム
- 是否涉及灯箱本体、电源、LED、PSE
- 尺寸、数量、画质、UV/コルトン素材

内部加价重点：

- 只做画面：物流简单
- 涉及本体/电气：必须个别确认 PSE、配线、安装责任
- 卷装运输需防压、防折

建议预备：

- 包装：采购价 5%-12%
- 电气相关风险预备：个别报价
- 日本安装：更换画面和灯箱本体安装分开报价

### 5.4 PVC 案内板 / プレート

计价基础：

- 厚度、尺寸、枚数
- UV 印刷、穴あけ、両面テープ
- 室内/屋外、耐候要求

内部加价重点：

- 板材边角破损风险
- 大尺寸板材超出宅配规格，需要路线便/チャーター

建议预备：

- 包装：采购价 8%-20%
- 日本配送：按尺寸决定，不可统一包邮
- 关税预留：塑料板/标识方向先按 0%-4% 估算，正式按 HS 确认

### 5.5 ホーロー / キッチンパネル / 板材类

计价基础：

- 张数、厚度、尺寸、表面处理
- 是否切割、开孔、边缘处理
- 住宅/店舗/厨房/浴室等使用环境

内部加价重点：

- 重、硬、易磕角
- 需要木架/木箱概率高
- 日本国内搬入和现场切割风险高

建议预备：

- 包装：采购价 15%-30%
- 国际运输：优先海运 LCL
- 日本配送：路线便/チャーター优先
- 现场施工：单独报价，不与材料价混在一起

## 6. 关税/税费内部预留规则

正式税率必须按 HS 编码确认。首版报价不能承诺固定关税。

| 品类 | 内部预留 |
|---|---|
| 窗帘/纺织成品 | 8%-12% 预留 |
| 亚克力/PVC/塑料标识 | 0%-4% 预留 |
| 印刷物/画面类 | 0%-5% 预留 |
| 家具/部分室内件 | 0%-5% 预留 |
| 地材/纺织地毯类 | 6%-9% 预留 |
| 电气/灯箱本体 | 0%-5% + PSE 个别确认 |
| WPC/复合建材 | 3%-8% 预留 |

进口消费税内部统一先按：

```text
进口消费税 = (CIF + 关税 + 其他应税进口费用) × 10%
```

## 7. 京建利润/服务费建议

| 案件类型 | 建议服务费/毛利 |
|---|---|
| 样品/小单 | 最低固定服务费 + 30%-45% |
| 标准材料供给 | 25%-35% |
| 施工業者复购 | 18%-28% |
| 大批量工程 | 12%-22% |
| 高风险/易碎/超长/电气 | 另加风险预备 5%-15% |

最低服务费建议：

- 小样/单件：JPY 5,000 - 15,000
- 一般询价整理：JPY 10,000 - 30,000
- 复杂项目：按项目管理费单列

## 9. Alibaba.com 扫描规则

Alibaba.com 是京建出海报价的主参考平台，但也不能只看最低价。每个 SKU 至少记录：

- 商品链接
- 店铺年限/回头率/成交记录
- 起订量
- 单价阶梯
- 是否支持定制
- 材质/厚度/规格
- 包装方式
- 发货地
- 贸易条款：EXW / FOB / CIF / DDP
- 报价币种：USD / RMB
- MOQ 对应价格
- 样品费
- 出口包装是否包含
- 是否可做日本向け规格确认
- 是否开票
- 是否支持图纸/AI/PDF 文件
- 是否可打样
- 预计生产周期
- 是否能做出口包装
- 评价中破损/色差/尺寸误差反馈

采购价判定：

```text
Alibaba.com基准采购价 = 取 3-5 家可出口定制供应商的中位价
```

1688 的作用：

```text
1688内销参考价 = 判断中国内销底价和供应链利润空间
```

不要用任何平台的最低价作为京建报价基础。
