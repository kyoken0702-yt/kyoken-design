# 京建サプライ 材料板块出海报价评估 v0.1

更新日：2026-06-10

## 1. 评估逻辑

京建材料板块不能只看单一采购价。每个品类按三层价格判断：

```text
1688 内销价
→ Alibaba.com 出口价
→ 京建日本可交付价
```

含义：

- 1688：判断中国内销底价和制造成本空间。
- Alibaba.com：判断出口供应商报价、MOQ、FOB/EXW、出口包装和外贸利润。
- 京建日本价：包含日本客户真正需要的到场、配送、测量、安装、售后和京建服务。

关键判断：

```text
如果 Alibaba.com 与 1688 的差价已经覆盖出口包装、FOB前费用、外贸服务和供应商利润，
京建仍需另算日本进口税费、日本国内配送、现场搬入、安装和自身服务费。
```

因此，Alibaba.com 价格不是最终成本，只是「出口侧参考价」。

## 2. 总体优先级

| 优先级 | 品类 | 判断 |
|---|---|---|
| A | オーダーカーテン | 最适合先做。轻、软、破损风险低，可形成套餐和安装服务差异。 |
| A | アクリルサイン / PVC看板 / バックライトフィルム | 适合施工業者、小店装修、看板店复购。需控制破损和尺寸。 |
| B | 吸音フェルトパネル | 轻、装饰性强，适合办公室/民泊/会议室。需确认防炎/环保表达。 |
| B | 壁紙 / 壁布 | 适合高毛利软装，但施工质量依赖日本现场。 |
| B- | WPC デッキ材 | 市场有吸引力，但重量、体积、日本配送和施工风险高。适合作为项目型，不适合小单随便卖。 |
| C+ | ホーロー / キッチンパネル | 客单价高，但重、硬、破损和现场切割风险高。需要严格项目报价。 |
| C / 个别确认 | 电气灯箱本体、户外大型看板 | PSE、电气工事、屋外施工、许可和安全责任复杂，第一阶段不应标准化。 |

## 3. 品类评估

### 3.1 オーダーカーテン

当前 Accio 对外基准：

- Plan A：¥15,800～
- Plan B：¥18,800～
- Plan C：¥24,800～
- 价格表区间：¥11,800 - ¥62,800

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | 可用来判断面料、加工、1.8倍褶皱、辅料的底价。 |
| Alibaba.com 出口价 | 可能已包含出口包装、外贸沟通和低 MOQ 上浮，但一般不含日本测量安装。 |
| 京建日本价 | 最大价值在「上门测量 + 标准简易安装 + 日语/中文沟通 + 现场确认」。 |

费用风险：

- 包装：低
- 海运/空运：低到中
- 关税/消费税：中，纺织品需按 HS 确认
- 日本配送：低
- 日本安装：中，但可标准化
- 售后风险：尺寸错误、色差、客户期待差异

结论：

窗帘是京建第一阶段最适合标准化的品类。Accio 当前价格可以保留，但必须明确：

```text
套餐可含现地採寸和既存レールへの標準簡易取付。
カーテンレール本体、新規レール取付、复杂施工、高所作業另算。
```

报价策略：

- 保留当前页面价格作为入口价。
- 对施工業者可做复购价。
- 对终端民泊/住宅可做含测量安装套餐。
- 1688/Alibaba 扫描重点不是找最低价，而是找稳定面料、稳定加工和低色差供应商。

### 3.2 アクリルサイン

当前 Accio 对外基准：

- Plan A：¥8,500～
- Plan B：¥18,500～
- Plan C：¥38,500～
- 价格表最高到 ¥88,800～

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | 亚克力加工内销价通常很低，适合判断加工底价。 |
| Alibaba.com 出口价 | 通常会把定制、文件沟通、出口包装、低 MOQ 计入报价。 |
| 京建日本价 | 价值在尺寸确认、图纸确认、安装方式、化粧ビス/孔位和现场照片判断。 |

费用风险：

- 包装：中，易刮花、易断角
- 国际物流：小件可空运，大件需木架
- 日本配送：中，大尺寸容易超规
- 日本安装：中，打孔和下地确认要另算
- 售后风险：色差、划伤、尺寸误差、孔位错误

结论：

适合做 B2B 小批量高毛利品类，尤其是办公室、餐饮店、美容室、民泊导视。  
但报价必须把「材料供给」和「现场安装」分开。

报价策略：

- 小件保留入口价。
- 大件必须案件报价。
- 超过一定尺寸需木架/路线便/现场安装确认。

### 3.3 バックライトフィルム / ライトボックス画面

当前 Accio 对外基准：

- Plan A：¥4,500/㎡～
- Plan B：¥15,800～
- Plan C：¥38,000～

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | UV 打印/灯箱片成本可参考，但内销包装通常不足。 |
| Alibaba.com 出口价 | 可能包含卷筒包装和外贸文件沟通。 |
| 京建日本价 | 价值在データ確認、尺寸确认、既存灯箱适配和更换施工。 |

费用风险：

- 包装：低到中，重点防折、防压
- 国际物流：低
- 关税：低到中，需按材质确认
- 日本配送：低
- 日本安装：低到中，如果只是更换画面容易标准化
- 电气风险：如果涉及灯箱本体/LED/PSE，风险立刻升高

结论：

只做画面/フィルム是好品类；做灯箱本体是另外一个高风险品类。

报价策略：

- 第一阶段主推「画面更换」「メニュー画面」「内照式フィルム」。
- 灯箱本体、电源、LED、PSE 统一个别确认。

### 3.4 PVC看板・プレート

当前 Accio 首页定位：

- 案内板・駐車場サイン・低コスト看板

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | PVC 板材和 UV 印刷底价低，适合做成本锚点。 |
| Alibaba.com 出口价 | 会包含低 MOQ、外贸沟通、包装和出口利润。 |
| 京建日本价 | 价值在小批量定制、照片确认、安装方式、孔位/テープ/屋外耐候确认。 |

费用风险：

- 包装：中，大板易弯折和破角
- 国际物流：中，大尺寸按体积计费
- 日本配送：中，超尺寸后费用明显上升
- 安装：低到中
- 售后风险：板材厚度、翘曲、色差、户外耐候期待

结论：

适合做施工業者/店舗小改装的小批量供应，但不宜承诺统一低价包邮。

报价策略：

- A4/A3/小尺寸可标准价。
- 大板、屋外、双面、孔位、厚板必须案件报价。

### 3.5 ホーローキッチンパネル

当前 Accio 对外基准：

- Plan A：¥35,000～
- Plan B：¥68,000～
- Plan C：¥120,000～

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | 可判断板材底价，但无法反映出口包装和破损风险。 |
| Alibaba.com 出口价 | 可能包含外贸包装，但不一定覆盖日本到场搬入和现场切割。 |
| 京建日本价 | 价值在项目型供货、尺寸确认、开孔、搬入和施工配合。 |

费用风险：

- 包装：高，木架/木箱概率高
- 海运：中到高
- 日本配送：高，板材重量/尺寸影响大
- 现场搬入：高
- 安装施工：高，切割、开孔、下地、厨房现场风险
- 售后风险：破损、边角磕碰、尺寸误差、现场不可施工

结论：

这是高客单价品类，但不适合作为轻松标准商品。应定位为项目型报价。

报价策略：

- 页面可保留套餐感，但报价必须「サイズ・下地・施工条件により個別見積」。
- 小单也要收项目管理费。
- 一定要把材料、配送、搬入、施工分开列。

### 3.6 吸音フェルトパネル

当前 Accio 对外基准：

- Plan A：¥12,800～
- Plan B：¥28,500～
- Plan C：¥78,000～

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | PET 吸音板内销价格有参考价值。 |
| Alibaba.com 出口价 | 通常可看到出口 MOQ、包装方式和板材规格。 |
| 京建日本价 | 价值在设计排版、墙面数量估算、日本现场安装或材料供给。 |

费用风险：

- 包装：低到中
- 国际物流：中，体积大但重量较轻
- 日本配送：中
- 安装：中，墙面定位和胶/ビス固定需确认
- 合规风险：防炎/环保表述不可乱承诺

结论：

适合办公室、会议室、民泊、录音室方向。比 WPC/ホーロー轻，风险更低。

报价策略：

- 可以作为中优先级产品。
- 不要强说防炎认证，写「案件ごとに確認」。
- 设计排版可作为京建增值服务。

### 3.7 壁紙 / 壁布

当前 Accio 定位：

- ホテル客室・エントランス
- 商業施設・レストラン
- シームレス、通気性、防カビ等卖点

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | 能判断壁布/壁纸印刷底价。 |
| Alibaba.com 出口价 | 更适合判断出口供应、卷装包装和 MOQ。 |
| 京建日本价 | 价值在尺寸测量、图案排版、施工安排和损耗控制。 |

费用风险：

- 包装：低
- 国际物流：低到中
- 日本配送：低
- 安装：高，施工质量高度依赖日本现场
- 售后风险：色差、拼缝、基层问题、损耗

结论：

材料本身适合出海，但利润成败在日本施工。  
如果只卖材料，售后争议少；如果含施工，需要强现场管理。

报价策略：

- 第一阶段建议「材料 + 图案排版 + 施工另算」。
- 商业空间可做项目报价。

### 3.8 WPC デッキ材

当前 Accio 对外基准：

- Plan A：¥14,800/㎡～
- Plan B：¥18,800/㎡～
- Plan C：¥24,800/㎡～
- 价格表区间：¥9,800/㎡～ 到 ¥24,800/㎡

三层价格判断：

| 层级 | 判断 |
---|---|
| 1688 内销价 | 能判断板材底价，但不能代表日本可交付成本。 |
| Alibaba.com 出口价 | 很可能已体现出口包装、FOB前费用、外贸利润、MOQ，但不含日本最后一公里和施工。 |
| 京建日本价 | 最大成本在海运、日本路线便/チャーター、现场搬入、根太/基础/施工。 |

费用风险：

- 包装：中
- 海运：高，重且体积大
- 关税/消费税：中
- 日本配送：高，长件重货最后一公里贵
- 施工：高，下地、水平、排水、基础、切割都影响报价
- 售后风险：变形、色差、热胀冷缩、客户误以为含基础施工

结论：

WPC 有外贸价格优势，但不适合用「小件商品」思维卖。  
它应该是项目型材料，适合 10㎡ 以上，最好由施工業者或京建合作施工队消化现场问题。

报价策略：

- Accio 当前页面价可作为材料入口价。
- 日本国内配送、根太、端部材、现场施工必须另算。
- 小面积反而不一定便宜，因为配送和施工固定成本摊不掉。

## 4. 可做性排序

### 第一阶段：立刻可做

1. オーダーカーテン
2. アクリルサイン
3. バックライトフィルム / メニュー画面
4. PVC案内板

特点：

- 小批量可做
- 物流可控
- 客户容易通过照片、尺寸、数量询价
- 京建能通过沟通和现场确认创造价值

### 第二阶段：项目型可做

1. 吸音フェルトパネル
2. 壁紙 / 壁布
3. WPC デッキ材
4. ホーローキッチンパネル

特点：

- 客单价更高
- 但现场条件影响大
- 需要材料价、配送价、施工价分开

### 暂不标准化

1. 灯箱本体
2. 大型户外看板
3. 电气相关产品
4. 需要认证承诺的产品

理由：

- PSE、防炎、屋外施工、电気工事、许可和安全责任复杂
- 可以个别接，但不做标准套餐

## 5. 结论

京建最适合的方向不是简单代购，而是：

```text
中国出口供应链价格优势
+ 日本现场测量/配送/安装能力
+ 华人/日本施工業者可理解的沟通
+ 案件制报价管理
```

Accio 当前总结出的品类方向总体是对的。  
要注意的不是“品类错”，而是每个品类必须分清：

- 哪些可标准价
- 哪些只能作为起步价
- 哪些必须案件报价
- 哪些必须把施工和配送另算
