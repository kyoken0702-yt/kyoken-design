## 2. 平台特征

### 楽天市場

特点：

- SKU 多，价格带宽。
- 搜索结果常混入样品、配件、小尺寸。
- 对窗帘、壁纸、吸音板这类软装价格竞争很充分。
- WPC 和ホーロー这类大件，价格带能反映日本零售端成本压力。

对京建意义：

- 适合判断日本市场最高/中位价格。
- 不适合用最低价比较京建，因为最低价多为样品、小件、配件。

### Yahoo!ショッピング

特点：

- 价格相对更贴近日常电商销售。
- 搜索页样本较干净，但仍混入小件。
- 有 PayPay/促销影响，实际成交价可能波动。

对京建意义：

- 适合作为日本普通消费者价格参考。
- 如果京建价格高于 Yahoo 很多，必须用“测量/安装/项目管理”解释。

### Amazon.co.jp

特点：

- 搜索结果噪音最大。
- 容易混入配件、多件套、工具、商业大宗、非同类商品。
- 对标准品有参考价值，对定制品和工程材料参考价值较弱。

对京建意义：

- 只适合观察是否有极低标准品冲击。
- 不适合作为定制工程品的主要定价基准。

## 3. 对京建各品类的具体判断

### 3.1 窗帘

Accio 当前价：

- Plan A：¥15,800～
- Plan B：¥18,800～
- Plan C：¥24,800～

市场对比：

- 楽天/Amazon 的中位和 P75 说明，日本电商上窗帘标准品非常多。
- Yahoo 上大量低价窗帘可能是成品尺寸或低规格，不等同于京建的上门测量和安装套餐。

判断：

```text
如果京建窗帘只卖材料，价格优势不明显。
如果京建包含现地採寸 + 既存レールへの標準簡易取付，则价格有合理性。
```

建议：

- 页面必须继续强调套餐服务范围。
- 不要和电商成品窗帘比最低价。
- 目标客户应是民泊、店舗、施工業者批量换装，而不是单个家庭买一副窗帘。

### 3.2 WPC人工木デッキ

Accio 当前价：

- ¥9,800-¥24,800/㎡

市场对比：

- 楽天/Yahoo 搜索结果中位大致在 ¥19,700-¥27,300 左右。
- 京建的 ¥9,800/㎡ 起步价有价格吸引力。
- 但市场平台的价格可能是板材套装、局部套件、不同面积单位，不完全可比。

判断：

```text
WPC 是京建有明显价格故事的品类，但不能把配送和施工吞进材料价。
```

建议：

- 页面保留材料入口价。
- 日本配送、根太、端部材、现场施工必须另算。
- 面积越大越适合京建，10㎡以下小单不一定划算。

### 3.3 アクリルサイン

Accio 当前价：

- ¥8,500～ / ¥18,500～ / ¥38,500～

市场对比：

- 楽天/Yahoo 小型アクリルサイン价格很低。
- Amazon 搜索结果混入大量不同类型，参考价值较弱。

判断：

```text
小型标准牌不要硬打价格战。
京建要做定制LOGO、厚板、大尺寸、安装方式确认、施工業者复购。
```

建议：

- 小件作为入口价。
- 主推受付、店舗ロゴ、オフィスサイン。
- 打孔、化粧ビス、下地确认、安装另算。

### 3.4 バックライトフィルム

Accio 当前价：

- ¥4,500/㎡～
- ¥15,800～
- ¥38,000～

市场对比：

- 楽天/Yahoo 中位约 ¥11,836-¥13,420。
- 京建 ¥4,500/㎡ 起步价有吸引力。
- 大尺寸高精细画面可以做到 ¥15,800～ / ¥38,000～ 的项目价。

判断：

```text
只做灯箱画面/フィルム是好品类。
灯箱本体和电气部分不应标准化。
```

建议：

- 主打メニュー画面、既存ライトボックスの画面交換。
- 电气、LED、PSE、灯箱本体全部个别确认。

### 3.5 PVC看板・プレート

市场对比：

- 三个平台都有大量低价小牌。
- 中位大致 ¥3,500-¥7,600。

判断：

```text
PVC小牌是低价透明品，京建不能只卖一块板。
```

建议：

- 做民泊案内、店舗导视、停车场、多枚数组合。
- 用案件数量、设计数据、尺寸、安装方式做差异化。
- 大板和户外耐候要求必须案件报价。

### 3.6 ホーローキッチンパネル

Accio 当前价：

- ¥35,000～
- ¥68,000～
- ¥120,000～

市场对比：

- 日本平台中位约 ¥12,782-¥30,900。
- Yahoo P75 约 ¥40,557，接近京建 Plan A。
- Accio 价格更像项目价，不是普通单张板材价。

判断：

```text
ホーロー不能按电商单板比价。
京建必须把它定义为厨房墙面项目/大面积墙面项目。
```

建议：

- 明确含不含切割、开孔、配送、搬入、施工。
- 如果只是板材供给，价格要谨慎。
- 如果含尺寸确认和项目管理，Plan A/B/C 有解释空间。

### 3.7 吸音フェルトパネル

Accio 当前价：

- ¥12,800～
- ¥28,500～
- ¥78,000～

市场对比：

- 日本平台材料价不高。
- 楽天中位约 ¥3,900，Yahoo 中位约 ¥6,180，Amazon 中位约 ¥7,600。

判断：

```text
吸音板材料本身不稀缺，京建要卖空间解决方案。
```

建议：

- 不要单卖普通 PET 板。
- 主打会议室、办公室、录音室、民泊、影音室的设计排版和施工。
- 防炎/环保等表述案件确认，不承诺认证完备。

### 3.8 壁紙・壁布

市场对比：

- 乐天/Yahoo 普通壁纸价格透明。
- Amazon 噪音很大，不宜直接参考。

判断：

```text
壁纸材料不难买，价值在图案定制、尺寸排版、施工管理。
```

建议：

- 做商业空间、民泊、餐饮店特色墙。
- 材料价透明，施工和设计是利润来源。
- 不建议做普通壁纸低价销售。

## 4. 京建报价策略调整

### 可标准化入口价

- 窗帘：保留 Accio 价，但绑定测量/标准安装价值。
- バックライトフィルム：保留 ¥4,500/㎡～ 作为强入口。
- WPC：保留 ¥9,800/㎡～ 作为材料入口，但必须注明配送/施工另算。

### 应案件报价

- ホーローキッチンパネル
- WPC 大面积/施工
- 壁紙・壁布施工
- 吸音板空间设计
- 大尺寸亚克力/PVC

### 不应打价格战

- 小型 PVC 牌
- 小型アクリル牌
- 普通壁纸
- 普通吸音板
- 电商成品窗帘

## 5. 最重要结论

日本电商平台已经证明：

```text
标准品价格非常透明。
京建不能只做“从中国买便宜材料”。
```

京建真正的报价逻辑应该是：

```text
中国供应链成本优势
+ 出口包装/物流整合
+ 日本最后一公里配送
+ 测量/安装/施工确认
+ 中日双语沟通
+ 施工業者背后的供货伙伴
```

所以 Accio 当前产品方向大体成立，但所有页面价格都要分清：

- 材料起步价
- 含测量安装套餐价
- 项目报价
- 大件配送/复杂施工另算
