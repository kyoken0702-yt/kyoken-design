# 京建サプライ (Kyoken Supply) VI 展開仕様
**物流・パッケージングに関するブランドガイドライン**

B2Bサプライチェーンとして、物流段階での「清潔感」と「信頼感」を演出するための視覚展開案です。

---

## 1. 配送用ダンボール (Shipping Box)
*   **ベース：** クラフト（茶）またはホワイト。
*   **印刷位置：** 側面の2箇所にメインロゴを印刷。
*   **情報記載：** 「FRAGILE（取扱注意）」および「KYOKEN SUPPLY B2B DIRECT」の文字をネイビーで配置。

## 2. ブランドロゴ入りテープ (Branded Tape)
*   **カラー：** ホワイトベースにネイビーのロゴ、またはネイビーベースにホワイトのロゴ。
*   **デザイン：** ロゴを30cm間隔でリピート配置。
*   **効果：** 梱包の密封性を高めると同時に、開封前からブランドを認知させます。

## 3. 納品書・送り状 (Delivery Note / Shipping Label)
*   **ヘッダー：** 右上に公式ロゴを配置。
*   **アクセントカラー：** 枠線や見出しに工業用ネイビー（#1e293b）を使用。
*   **フォント：** Noto Sans JP（モダンで読みやすいサンセリフ体）。

## 4. ケアマーク (Care Instructions Symbols)
*   **アイコン：** JIS規格準拠のアイコンを使用。
    *   カッター使用注意（壁紙・フィルム用）
    *   角当て注意（ホーロー・看板用）
